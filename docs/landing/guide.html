p4v-tui 사용 가이드

설치·연결부터 트리 파일 작업, 체인지리스트·서브밋, 디프·히스토리, 리졸브·머지, 검색·이동, 연결 회복력, 좁은 화면, 설정까지 — p4v-tui 의 모든 기능을 실제 화면 스크린샷과 함께 챕터별로 안내합니다. 각 페이지 아래에는 이전·다음 챕터 링크가 있습니다. 처음이라면 1. 시작 · 핵심 개념부터 읽으세요. 빠르게 둘러보려면 랜딩 페이지를 먼저 보세요.
